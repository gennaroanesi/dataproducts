Principal Components / Exploratory Factor Analysis on the Wine database


This shiny app intends to do a Principal Components or an Exploratory Factor Analysis on the Wine dataset. These tecnhiques are statistical methods to reduce the number of variables in a database. In this app, you can select either of the two methods, as well as a rotation method and the desired number of factors. Outputs are: an analysis summary, biplots of the model as well the resulting factor loadings. The app uses the sqldf library for the selection of variables, the psych library to compute the components/factors and the GPARotation library to compute the rotations.
